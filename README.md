# Installation

```sh
curl -Lsk https://raw.githubusercontent.com/nowak0x01/lastwhisper/main/lastwhisper.sh -o ./lastwhisper.sh
chmod +x ./lastwhisper.sh
./lastwhisper.sh --usage
```
